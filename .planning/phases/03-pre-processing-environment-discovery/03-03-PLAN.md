  <behavior>
    - scanEnvironment(cwd) produces a valid EnvironmentSnapshot
    - Version detection: runs `execFileSync('claude', ['--version'])`, parses "X.Y.Z" from output, returns { version, version_known: true, compatible: true/false }
    - If claude CLI not found or output unparseable: returns { version: 'unknown', version_known: false, compatible: false }
    - Compatible range: 2.1.0 through 2.1.99 (known tested versions)
    - Settings: reads ~/.claude/settings.json, {cwd}/.claude/settings.json, {cwd}/.claude/settings.local.json. Returns parsed JSON or null if file missing/invalid
    - Plugins: reads ~/.claude/plugins/installed_plugins.json, cross-references with enabledPlugins from user settings. For each plugin, scans cache dir for capabilities (commands/, skills/, hooks/, agents/ subdirs)
    - Skills: scans ~/.claude/skills/ and {cwd}/.claude/skills/ for directory names. Reports scope ('user' or 'project')
    - Rules: scans {cwd}/.claude/rules/ for directory names. Reports scope 'project'
    - Hooks: extracts hook events from settings.json hooks config at all 3 scopes. Reports event name, scope, and type (command/agent)
    - CLAUDE.md: checks existence of ./CLAUDE.md, ./.claude/CLAUDE.md, ~/.claude/CLAUDE.md. Reports path and exists boolean
    - Ecosystems: checks for .planning/ dir (GSD), ~/.claude/skills/ containing 'cog' (Cog ecosystem). Returns string array of detected ecosystem names
    - Writes snapshot atomically to paths.environmentSnapshot using write-file-atomic
    - All filesystem operations are wrapped in try-catch -- missing dirs/files never crash the scanner
  </behavior>

    Implementation -- use `Promise.all` to parallelize independent reads:

    1. **detectClaudeCodeVersion()**: `execFileSync('claude', ['--version'], { timeout: 3000, encoding: 'utf-8', stdio: ['pipe', 'pipe', 'pipe'] })`. Parse with `/^(\d+\.\d+\.\d+)/` regex. Compare using `compareSemver()` helper (split on '.', compare numeric parts). Return `{ version, version_known, compatible }`.

    2. **readSettingsSafe(filePath: string): Promise<unknown | null>**: Try `readFile(filePath, 'utf-8')` then `JSON.parse()`. Return parsed object or null on any error.

    3. **discoverPlugins(home: string, enabledPluginNames: string[]): Promise<EnvironmentSnapshot['installed_tools']['plugins']>**: Read `~/.claude/plugins/installed_plugins.json`. Parse as JSON array. For each plugin entry, check if name is in enabledPluginNames. Scan `~/.claude/plugins/cache/{marketplace}/{pluginName}/{version}/` for subdirs (commands, skills, hooks, agents). Return array. On error, return [].

    4. **discoverSkills(home: string, cwd: string): Promise<...>**: Read directory names in `~/.claude/skills/` (scope: 'user') and `{cwd}/.claude/skills/` (scope: 'project'). Filter to directories only using `readdir` with `withFileTypes: true`. Return array of { name, scope }. On error, return [].

    5. **discoverRules(cwd: string): Promise<...>**: Read directory names in `{cwd}/.claude/rules/`. Scope always 'project'. Return array. On error, return [].

    6. **discoverHooks(userSettings: unknown, projectSettings: unknown, localSettings: unknown): ...**: Extract hooks config from each settings object (if it has a `hooks` property which is a Record<string, HookDef[]>). For each event+hook, record { event, scope, type }. Return flat array.

    7. **discoverClaudeMd(home: string, cwd: string): Promise<...>**: Check existence of `{cwd}/CLAUDE.md`, `{cwd}/.claude/CLAUDE.md`, `{home}/.claude/CLAUDE.md` using `access(path, constants.F_OK)`. Return array of { path, exists }.

    8. **detectEcosystems(cwd: string, home: string): Promise<string[]>**: Check `.planning/` dir exists (→ 'gsd'). Check `~/.claude/skills/` for any dir containing 'cog' (→ 'cog'). Return array.

    9. **compareSemver(a: string, b: string): number**: Split on '.', compare each numeric part. Return -1, 0, or 1.

    Orchestration in `scanEnvironment`:
    - First read all 3 settings files in parallel
    - Extract enabledPlugins from user settings (if present, it's an array of strings or objects with name property)

    - Construct EnvironmentSnapshot, validate with `environmentSnapshotSchema.parse()`
    - Write atomically: `await ensureInit(); await writeFileAtomic(paths.environmentSnapshot, JSON.stringify(snapshot));`
    - Return validated snapshot
