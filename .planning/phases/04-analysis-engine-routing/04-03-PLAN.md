    **Step 2: Create src/analysis/trigger.ts**

    Imports:
    ```typescript
    import { readCounter, resetCounter } from '../storage/counter.js';
    import { loadConfig } from '../storage/config.js';
    import { paths, ensureInit } from '../storage/dirs.js';
    import { preProcess } from './pre-processor.js';
    import { scanEnvironment } from './environment-scanner.js';
    import { analyze } from './analyzer.js';
    import type { AnalysisResult } from '../schemas/recommendation.js';
    import writeFileAtomic from 'write-file-atomic';
    import { readFile } from 'node:fs/promises';
    import { lock } from 'proper-lockfile';
    ```

    Export `writeAnalysisResult(result: AnalysisResult): Promise<void>`:
    - `await ensureInit()`
    - `await writeFileAtomic(paths.analysisResult, JSON.stringify(result, null, 2))`

    Export `runAnalysis(cwd: string): Promise<AnalysisResult>`:
    - Call `const summary = await preProcess()`
    - Call `const snapshot = await scanEnvironment(cwd)`
    - Call `const result = analyze(summary, snapshot)`
    - Call `await writeAnalysisResult(result)`
    - Return `result`

    Export `checkAndTriggerAnalysis(cwd: string): Promise<boolean>`:
    - `const config = await loadConfig()`
    - If `!config.analysis.enabled` return false
    - `const counter = await readCounter()`
    - If `counter.total < config.analysis.threshold` return false
    - If `counter.last_analysis`:
      - Parse timestamp, compare to `Date.now()`
      - `const COOLDOWN_MS = 60_000` (1 minute)
      - If within cooldown, return false
    - Call `await runAnalysis(cwd)`
    - Update counter's `last_analysis` field. Do this by:
      1. Lock the counter file (same pattern as incrementCounter)
      2. Read current counter
      3. Set `last_analysis` to current ISO timestamp
      4. Reset `total` to 0 and `session` to `{}`
      5. Write atomically
      6. Unlock
    - Return true
    - Wrap runAnalysis + counter update in try-catch: if runAnalysis fails, do NOT reset counter (preserve data for next trigger attempt). Re-throw or return false with logged error.

    Note: The counter update needs its own lock + atomic write to safely reset. Create a helper `resetCounterWithTimestamp(): Promise<void>` that:
    1. Ensures counter file exists
    2. Locks the counter file
    3. Writes `{ total: 0, session: {}, last_analysis: new Date().toISOString(), last_updated: new Date().toISOString() }`
    4. Unlocks
