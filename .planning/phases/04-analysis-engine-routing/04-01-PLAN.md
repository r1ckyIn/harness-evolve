    **Step 1: Create src/analysis/classifiers/repeated-prompts.ts**

    Export `classifyRepeatedPrompts(summary, snapshot, config): Recommendation[]`.

    Logic:
    - Iterate `summary.top_repeated_prompts`
    - Skip entries where `entry.count < config.thresholds.repeated_prompt_min_count` (default 5)
    - Compute word count: `entry.prompt.split(/\s+/).length`
    - Skip entries where wordCount > 50 (those are long prompts, handled by long-prompts classifier)
    - Determine confidence:
      - HIGH: `count >= config.thresholds.repeated_prompt_high_count (10)` AND `sessions >= config.thresholds.repeated_prompt_high_sessions (3)`
      - MEDIUM: otherwise (already passed the min threshold)
    - Create Recommendation with:
      - `id`: `rec-repeated-${index}` (use loop index for deterministic IDs in tests)
      - `target`: `'HOOK'`
      - `pattern_type`: `'repeated_prompt'`
      - `title`: `Repeated prompt: "${entry.prompt}"` (truncate prompt to 60 chars with ellipsis if longer)
      - `description`: `This prompt has been used ${entry.count} times across ${entry.sessions} sessions. Consider creating a hook or alias to automate this.`
      - `evidence`: `{ count: entry.count, sessions: entry.sessions, examples: [entry.prompt] }`
      - `suggested_action`: `Create a UserPromptSubmit hook that detects "${entry.prompt}" and auto-executes the intended action.`

    **Step 2: Create src/analysis/classifiers/long-prompts.ts**

    Export `classifyLongPrompts(summary, snapshot, config): Recommendation[]`.

    Logic:
    - Iterate `summary.long_prompts`
    - Skip entries where `entry.count < config.thresholds.long_prompt_min_count (2)` OR `entry.length < config.thresholds.long_prompt_min_words (200)`
    - Determine confidence:
      - HIGH: `count >= config.thresholds.long_prompt_high_count (3)` AND `length >= config.thresholds.long_prompt_high_words (300)`
      - MEDIUM: otherwise
    - Create Recommendation with:
      - `id`: `rec-long-${index}`
      - `target`: `'SKILL'`
      - `pattern_type`: `'long_prompt'`
      - `title`: `Long repeated prompt (${entry.length} words, ${entry.count}x)`
      - `description`: `A ${entry.length}-word prompt has been used ${entry.count} times. Consider converting it to a reusable skill.`
      - `evidence`: `{ count: entry.count, examples: [entry.prompt_preview] }`
      - `suggested_action`: `Create a skill in .claude/skills/ that encapsulates this prompt as a reusable workflow.`

    **Step 3: Create src/analysis/classifiers/permission-patterns.ts**
